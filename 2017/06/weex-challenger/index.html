发布

因为玩过 React Native，打包成一个应用本是一件不复杂的事情，然而在 weex 中就没那么规范了。

weex-toolkit 虽然已经集成了 weexpack 的功能，但二者初始化出来的项目完全不同。也就是 weex-toolkit init != weexpack create，以致于后面的 weexpack platform 也无法继续。暂时没空折腾了。。。
